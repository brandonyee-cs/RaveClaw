# RaveClaw

> An OpenClaw skill for NYC rave curation. Upload a flyer, get a ranked lineup scored by cultural relevance and ticket price.

---

## External resources

- [Anthropic Claude API](https://docs.anthropic.com) — vision (flyer parsing) and LLM (ACI scoring)
- [Exa](https://exa.ai) — semantic web search for ticket prices
- [python-telegram-bot](https://python-telegram-bot.org) — Telegram interface
- Claude (claude.ai) — used for architecture review and code structure
